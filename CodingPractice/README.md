# 코딩 연습 - 개체 만들기

교안의 예제 코드를 직접 작성하고 실행하는 프로젝트입니다.

> **참고**: 과제 사이에 변수 이름이 겹칠 수 있습니다. 
이전 과제의 코드를 지우거나 주석 처리한 후 다음 과제를 작성하세요.

## 과제 1: 간단한 클래스와 개체

`Dog` 클래스를 정의하고 개체를 생성하여 사용하는 코드를 작성하세요.

- `Dog` 클래스에 `Name`(string), `Age`(int) 필드와 `Bark()` 메서드를 정의
- `myDog` 개체를 생성하고 이름을 "초코", 나이를 3으로 설정
- `Bark()` 메서드를 호출하고 이름과 나이를 출력

**실행 결과**

```
초코: 멍멍!
이름: 초코, 나이: 3살
```

## 과제 4: 인스턴스 메서드와 필드

필드에 접근하는 인스턴스 메서드를 작성하세요.

- `Counter` 클래스에 `Count`(int) 필드와 `Increment()`, `Reset()` 메서드를 정의
- `Increment()`는 `Count`를 1 증가시키고 현재 카운트를 출력
- `Reset()`은 `Count`를 0으로 초기화하고 초기화 메시지를 출력
- `Increment()`를 3번 호출한 후 `Reset()`을 호출

**실행 결과**

```
현재 카운트: 1
현재 카운트: 2
현재 카운트: 3
카운트가 초기화되었습니다.
```

## 과제 5: 정적 멤버와 인스턴스 멤버

정적 멤버와 인스턴스 멤버를 함께 사용하는 `GameManager` 클래스를 작성하세요.

- 정적 멤버: `TotalPlayers`(static int), `ShowStatus()` 정적 메서드
- 인스턴스 멤버: `PlayerName`(string), `Greet()` 메서드
- 정적 멤버는 `클래스명.멤버`로, 인스턴스 멤버는 `개체명.멤버`로 접근

**실행 결과**

```
[정적] 총 플레이어 수: 10
[인스턴스] 안녕하세요, 홍길동님!
```
